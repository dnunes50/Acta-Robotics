Saldo Inicial por Conta BancáriaBase para cálculo do saldo atual
Informe o saldo de cada conta bancária na data do primeiro lançamento no extrato. O saldo atual de cada conta será calculado automaticamente: Saldo Inicial + soma de todos os lançamentos.
SQL necessário no Supabase
Execute esses comandos no Supabase → SQL Editor antes de usar esta tela: 
ALTER TABLE extratos ADD COLUMN IF NOT EXISTS banco TEXT DEFAULT 'Inter';

CREATE TABLE IF NOT EXISTS saldos_iniciais (
  banco TEXT PRIMARY KEY,
  saldo_inicial NUMERIC NOT NULL DEFAULT 0,
  data_referencia DATE,
  updated_at TIMESTAMP DEFAULT NOW()
);
